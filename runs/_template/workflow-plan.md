## Phase 1: Problem Decomposition

**Goal:** Reach atomic truth of the problem — first principles + 5 Whys.

**Gate criteria:**
- [ ] Root cause feels irreducible
- [ ] Every terminal why cited with at least one external source (or flagged)
- [ ] Job executor is a person, not an organization
- [ ] Problem statement is solution-agnostic
- [ ] Problem statement validated with 3 cited sources
