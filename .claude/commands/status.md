# /status — Create a Status Update

Generate a status update for stakeholders.

## Instructions

1. Ask: "What project or work needs a status update?"
2. Gather:
   - Progress since last update
   - Current status (On Track / At Risk / Blocked)
   - What's coming next
   - Any risks or blockers
   - Any asks from stakeholders
3. Generate using @.claude/templates/TEMPLATE_status-update.md
4. Apply guidance from @.claude/skills/status-updates/SKILL.md

## Key Points

- Answer: What happened? So what? What's next?
- Lead with the headline
- Focus on outcomes, not activities
- Surface risks early

## Status Indicators

- **On Track**: Progressing as planned
- **At Risk**: May miss targets without intervention
- **Blocked**: Cannot progress until resolved
- **Complete**: Done

## Output

A formatted status update ready to send.
